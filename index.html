sqrrl is a Big Data company that provides products for Apache Accumulo.

With sqrrl, Big Data becomes secure, scalable, and adaptable to a wide variety of data environments and analytical requirements

Apache Accumulo

Apache Accumulo™ is a sorted, distributed key/value store. It handles large amounts of structured, semi-structured, and unstructured data as a robust, scalable, and real-time data storage and retrieval system. It can be integrated with any Hadoop deployment.

Apache Accumulo has three unique technical advantages over other comparable database solutions:

  • Security: Fine-grained security controls allow organizations to control data at the cell-level and promote a data-centric security model without degrading performance. Download sqrrl’s Cell-Level Security Data Sheet here.
  • Scalability: Accumulo is proven to operate and perform at massive scale (i.e., petabytes of data) with low administrative overhead
  • Adaptability: sqrrl’s solutions enable users to move beyond Hadoop batch processing and conduct a wide variety of real-time analyses, including information retrieval, statistical, graph, and visualizations across diverse data environments.

For more information go to http://accumulo.apache.org.

Product

Acorn builds upon the open source version of Apache Accumulo to include additional features that make Big Data secure, scalable and adaptable. These features focus on ease of use, security extensions, and advanced analytic capabilities.
